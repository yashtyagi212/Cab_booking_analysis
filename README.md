# Cab Booking Analysis Dashboard

## Project Overview
This project analyzes cab booking data to identify booking patterns, peak demand hours, busiest weekdays, and top-performing operational bases. The dataset was cleaned using Python, analyzed with SQL, and visualized using Power BI.

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL / MySQL
- Power BI
- Excel

## Project Workflow
1. Data Cleaning using Python  
2. Data Import and Querying using SQL  
3. Dashboard Creation using Power BI  
4. Insight Generation

## Key Insights
- Peak booking demand observed during evening hours (5 PM to 6 PM)
- Friday recorded the highest number of bookings
- Base B02598 handled the maximum bookings
- Booking demand gradually increased during weekdays

## Dashboard Features
- Total Bookings KPI Card
- Bookings by Hour Line Chart
- Bookings by Weekday Bar Chart
- Bookings by Base Comparison
- Interactive Filters (Month, Weekday, Base)
